# 上线运维清单

## 账号和授权

- Cloudflare：需要登录并允许创建 Worker、D1、R2、Secrets。
- DeepSeek：需要可用 API Key 和余额。
- 微信支付：需要商户号、AppID、API v3 密钥、商户私钥、证书序列号、支付回调域名。
- 短信：建议腾讯云短信，需签名、模板、密钥。
- 邮件：可用 Resend、腾讯云邮件或阿里云邮件，需发送域名或发件身份。

## 支付安全

- 不使用个人收款码监听。
- 不使用免签约码支付。
- 不让 AI 判断支付成功。
- 只接受微信支付 API v3 验签后的回调。
- 回调必须幂等：同一个 `out_trade_no` 只能入账一次。
- 现金余额和赠送余额分开，赠送余额不可提现、不可退款。

## 成本保护

- 单次生成必须先预估最高扣费。
- 余额不足不调用 DeepSeek。
- DeepSeek 超时或返回格式错误不扣费。
- 同一任务重试要复用 job id，避免重复扣费。
- 上传文件限制 10MB 起步，默认 7 天清理。
- 短信验证码要限频：同手机号、同 IP、同用户都要限制。

## 隐私保护

- 日志中不得出现完整手机号、邮箱、微信支付流水、API Key、上传文件内容。
- R2 文件 key 使用随机 UUID，不使用原始文件名作为路径。
- 管理后台每次余额调整都写审计日志。
- 用户导出的图默认不含水印、学校、作者、邮箱、本地路径。

## 绘图库边界

- 原始 `/Users/wi/Documents/Study/17 akun` 只读。
- 不复制原始代码并删除署名。
- 不发布 `.pyc`、`.DS_Store`、`.p`、`.mat`、`.fig`、`.docx`、`.xlsx`、`.vsd`。
- `npm run audit:plot-library` 只做盘点，不修改源目录。
